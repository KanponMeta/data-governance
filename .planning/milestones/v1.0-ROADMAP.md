# Milestone v1.0: MVP

**Status:** ✅ SHIPPED 2026-05-12
**Phases:** 1-6
**Total Plans:** 37

## Overview

Complete data governance platform MVP delivering Go-based asset execution engine, field-level lineage, schema evolution tracking, governance workflow with RBAC, and React SPA with ConnectRPC API.

## Phases

**Goal**: 平台以单二进制运行，具备健康的 PostgreSQL 存储层、版本化迁移、追加式事件日志、用户认证和稳定版本化的连接器接口
**Depends on**: None
**Plans**: 5 plans

Plans:
- [x] 01-01-PLAN.md: Go module bootstrap, ent schema, atlas migrations, PostgreSQL setup
- [x] 01-02-PLAN.md: Storage interface, event writer, append-only log, RFC 7807 errors

**Details:** Phase 1 delivered the platform foundation with PostgreSQL storage via ent, Atlas migrations, append-only event log with RLS protection, JWT authentication with invite flow, and stable v1.0.0 connector ABI.

## Milestone Summary

**Key Decisions:**
- Asset-centered model (vs task-centered) — Dagster proven mental model
- Pure Go core, no Python runtime — operational simplicity
- Field-level lineage as first-class feature — primary differentiation from Dagster
- hashicorp/go-plugin for connector extensibility — process isolation, language-agnostic

**Issues Deferred:**
- Phase 6 quality trend/alerts stubs (ent queries not wired) — INT-05 QualityEvaluator wiring gap
- Phase 6 AdminService Unimplemented (ColumnPolicy schema missing) — INT-01 GovernanceWorkflow wiring gap
- main.go wiring gaps (GovernanceWorkflow, Enforcer, AuthMW not wired) — bootstrap dependency injection incomplete
- Phase 4 EXPLAIN ANALYZE capture (needs live PostgreSQL) — human UAT deferred
- Phase 5 Snowflake/BigQuery/webhook live-system tests — requires real credentials

**Technical Debt:**
- QualityEvaluator not wired to Executor (INT-05) — quality rules never evaluated
- GovernanceWorkflow not wired to API deps (INT-01) — governance UI non-functional
- Casbin Enforcer not wired (INT-02) — RBAC checks always fail open
- GovernanceGatingEnabled not set (INT-04) — executor gate is decorative
