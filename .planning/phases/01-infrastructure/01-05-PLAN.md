## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Internet to API | All HTTP requests reach the chi router; middleware enforces auth before reaching protected handlers |
| Authenticated User to Admin Routes | RequireRole("admin") gates invite issuance |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Information Disclosure | Auth error responses | mitigate | All login failures return identical 401 problem+json with title "Invalid credentials" regardless of which check failed (prevents user enumeration). Achieved by mapping ErrInvalidCredentials uniformly in the handler. |
| T-05-02 | Repudiation | Auth events | mitigate | Login, register, invite, token-expired events appended to event_log with actor_id, IP, user-agent; event_log itself is append-only (Plan 02 RLS). |
| T-05-03 | DoS | Login | accept | Rate limiting deferred to Phase 5; Phase 1 single-machine dev usage bounds blast radius. |
| T-05-04 | EoP | Invite tokens | mitigate | Tokens generated via crypto/rand 32 bytes; stored as sha256 hex; raw token returned only once; expiry default 72h; single-use via accepted_at set in transaction. |
| T-05-05 | Injection | Request bodies | mitigate | json.Decoder.DisallowUnknownFields(); max body 1MB via http.MaxBytesReader; ent parameterizes queries. |
| T-05-06 | EoP | Admin invite endpoint | mitigate | RequireRole("admin") middleware on POST /v1/auth/invites; first user is bootstrapped as admin only when DB empty. |
| T-05-07 | Tampering | grpc stub | accept | Phase 1 stub returns canned JSON; no auth enforced on /grpc/* during Phase 1. Phase 2's connect-go replacement MUST add auth.Middleware to /grpc; documented in SUMMARY as a known technical debt. |

  - call `HashPassword`

- `Invite(ctx, adminID, email)`:
  - generate 32 random bytes via `crypto/rand.Read`
  - rawToken = `base64.RawURLEncoding.EncodeToString(bytes)`
  - sum := `sha256.Sum256([]byte(rawToken))`; tokenHash = `hex.EncodeToString(sum[:])`
  - persist InviteToken{token_hash:tokenHash, email, invited_by:adminID, expires_at:now+inviteTTL}
  - emit `EventTypeUserInvited` with `UserInvitedPayload{InviteID, Email, InvitedBy:adminID, ExpiresAt}`; ActorID=adminID
- `AcceptInvite(ctx, rawToken, password)`:
  - tokenHash = hex sha256 of rawToken (same algorithm as Invite)
  - inside WithTx: lookup InviteToken by token_hash. Map: missing→ErrInviteNotFound, accepted_at != nil→ErrInviteAlreadyUsed, expires_at <= now→ErrInviteExpired
  - HashPassword; create User with role=member, status=active
  - Update InviteToken.accepted_at = now
  - emit `EventTypeUserRegistered` after commit
