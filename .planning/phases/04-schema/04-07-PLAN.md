<objective>
Phase 4 acceptance criterion #1 ("lineage API"), #3 ("impact analysis API"), #4 ("schema timeline"), and #5 ("metadata via API") all require HTTP surfaces. Plans 04-02..04-06 land the storage and library code; this plan ships the REST endpoints and the OpenLineage export translator that make those libraries usable from outside the Go process.

Specifically, this plan delivers:

1. **Metadata mutation API** (D-17, META-03) — `PATCH /v1/assets/:name/metadata`, `PATCH /v1/assets/:name/columns/:col/metadata`, `GET /v1/assets/:name/metadata`. Backed by a new `internal/metadata` package that owns the `asset_metadata` ent entity reads/writes and emits `metadata.updated` events.
2. **Schema-change ack API** (D-10, META-02) — `POST /v1/schema/changes/:id/ack` and `GET /v1/schema/changes` for the META-05 timeline view, both protected by RequireRole("governance").
3. **Impact analysis REST** (D-19, D-20, LINE-06) — `GET /v1/lineage/impact` wraps the Wave 6 `impact.Analyze` library; depth-cap rejection (HTTP 400 above 25) and direction parameter validation enforced at the handler boundary.
4. **OpenLineage export** (D-18, LINE-01) — `GET /v1/lineage/export?format=openlineage` and the in-house translator at `internal/lineage/openlineage/translate.go`. Zero new go.mod entries — ThijsKoot/openlineage-go is explicitly NOT vendored (CLAUDE.md LOW credibility flag).
5. **Router wiring** — new routes mounted under the protected (JWT) group in `internal/api/router.go`; governance-only routes additionally guarded by `auth.RequireRole("governance")`.

This is the user-facing surface gate. Without this plan, every D-XX decision that requires a REST endpoint or HTTP-callable behavior is unreachable — that violates the scope-reduction prohibition. After this plan ships, all eight Phase 4 requirements (LINE-01/02/03/06, META-01/02/03/05) have a callable HTTP surface in addition to their library + capture path.

Wave-level placement: Wave 7. Depends on 04-04 (capture writers populating asset_edges/schema_versions/schema_changes), 04-05 (diff classifier producing schema_changes rows), 04-06 (impact.Analyze library). Cannot run earlier because every endpoint reads or writes data produced by those upstream plans.

       package metadata

       import (
           "context"
           "encoding/json"
           "errors"
           "fmt"
           "net/http"

           "github.com/go-chi/chi/v5"
           "github.com/google/uuid"
           "github.com/kanpon/data-governance/internal/auth"
           "github.com/kanpon/data-governance/internal/event"
       )

       const MaxTags = 64

       type Handler struct {
           store  *Store
           events event.Writer
       }
       func NewHandler(store *Store, events event.Writer) *Handler {
           return &Handler{store: store, events: events}
       }

       type patchBody struct {
           Description *string  `json:"description"`
           Owner       *string  `json:"owner"`
           Tags        *[]string `json:"tags"`
       }

       func (h *Handler) PatchAsset(w http.ResponseWriter, r *http.Request) {
           name := chi.URLParam(r, "name")
           h.patch(w, r, name, nil)
       }
       func (h *Handler) PatchColumn(w http.ResponseWriter, r *http.Request) {
           name := chi.URLParam(r, "name")
           col := chi.URLParam(r, "col")
           h.patch(w, r, name, &col)
       }
       func (h *Handler) Get(w http.ResponseWriter, r *http.Request) {
           name := chi.URLParam(r, "name")
           col := r.URL.Query().Get("column")
           var colPtr *string
           if col != "" { colPtr = &col }
           res, err := h.store.Get(r.Context(), name, colPtr)
           if err != nil { /* problem json */ return }
           writeJSON(w, http.StatusOK, res)
       }

       func (h *Handler) patch(w http.ResponseWriter, r *http.Request, name string, col *string) {
           var body patchBody
           if err := json.NewDecoder(r.Body).Decode(&body); err != nil { /* 400 */ return }
           if body.Tags != nil && len(*body.Tags) > MaxTags {
               writeProblem(w, http.StatusBadRequest, "tags_too_many",
                   fmt.Sprintf("tags exceeds %d entries", MaxTags))
               return
           }
           principal, _ := auth.PrincipalFromContext(r.Context())
           merge := r.URL.Query().Get("merge") == "true"
           // Load before-state for audit
           before, _ := h.store.Get(r.Context(), name, col)
           after, err := h.store.Put(r.Context(), PutInput{
               Asset: name, Column: col,
               Description: body.Description, Owner: body.Owner, Tags: body.Tags,
               SetBy: principal.UserID, Merge: merge,
           })
           if err != nil { /* problem */ return }
           // Emit event
           _ = h.events.Append(r.Context(), event.Event{
               Type: event.EventTypeMetadataUpdated,
               OccurredAt: time.Now().UTC(),
               ActorID: &principal.UserID,
               ResourceType: "asset_metadata",
               ResourceID: name,
               Payload: event.MetadataUpdatedPayload{
                   Asset: name, Column: col, ActorID: principal.UserID.String(),
                   BeforeDesc: before.Effective.Description, /* ...etc... */
                   AfterDesc: after.Description, AfterOwner: after.Owner, AfterTags: after.Tags,
                   Merge: merge,
               },
           })
           writeJSON(w, http.StatusOK, map[string]any{"effective": after})
       }

                       "asset": existing.Asset,
                       "reason": body.Reason,
                       "acknowledged_by": principal.UserID.String(),
                   },
               })

       package openlineage

       import (
           "context"
           "fmt"
           "time"

           "github.com/google/uuid"
           "github.com/kanpon/data-governance/internal/storage/ent"
           assetedge "github.com/kanpon/data-governance/internal/storage/ent/assetedge"
           columnedge "github.com/kanpon/data-governance/internal/storage/ent/columnedge"
           runent "github.com/kanpon/data-governance/internal/storage/ent/run"
       )

       const (
           Producer  = "https://github.com/kanpon/data-governance"
           SchemaURL = "https://openlineage.io/spec/2-0-2/RunEvent.json"
       )

       type RunEvent struct {
           EventType string      `json:"eventType"`
           EventTime string      `json:"eventTime"`
           Run       OLRun       `json:"run"`
           Job       OLJob       `json:"job"`
           Inputs    []OLDataset `json:"inputs"`
           Outputs   []OLDataset `json:"outputs"`
           Producer  string      `json:"producer"`
           SchemaURL string      `json:"schemaURL"`
       }
       type OLRun struct {
           RunID  string         `json:"runId"`
           Facets map[string]any `json:"facets,omitempty"`
       }
       type OLJob struct {
           Namespace string         `json:"namespace"`
           Name      string         `json:"name"`
           Facets    map[string]any `json:"facets,omitempty"`
       }
       type OLDataset struct {
           Namespace string         `json:"namespace"`
           Name      string         `json:"name"`
           Facets    map[string]any `json:"facets,omitempty"`
       }

       type Translator interface {
           TranslateRun(ctx context.Context, runID uuid.UUID) (RunEvent, error)
           TranslateAsset(ctx context.Context, asset string, since time.Time) ([]RunEvent, error)
       }

       type DefaultTranslator struct {
           ent       *ent.Client
           Namespace string // e.g. "platform.local"
       }

       func NewDefault(c *ent.Client, ns string) *DefaultTranslator {
           if ns == "" { ns = "platform" }
           return &DefaultTranslator{ent: c, Namespace: ns}
       }

       func (t *DefaultTranslator) TranslateRun(ctx context.Context, runID uuid.UUID) (RunEvent, error) {
           run, err := t.ent.Run.Query().Where(runent.IDEQ(runID)).Only(ctx)
           if err != nil {
               return RunEvent{}, fmt.Errorf("openlineage: load run %s: %w", runID, err)
           }
           if run.State != "succeeded" {
               return RunEvent{}, fmt.Errorf("openlineage: run %s state=%s, only succeeded runs export", runID, run.State)
           }
           // Inputs: POINT-IN-TIME edges as of run.StartedAt (D-15 soft-retire pattern).
           // OpenLineage events must reflect the lineage state at the time of the run, not now —
           // an edge retired between run completion and export must STILL appear in this run's
           // RunEvent. Predicate: first_seen_at <= run.StartedAt AND (superseded_at IS NULL OR superseded_at > run.StartedAt).
           runStarted := run.StartedAt
           edges, _ := t.ent.AssetEdge.Query().
               Where(
                   assetedge.ToAssetEQ(run.AssetName),
                   assetedge.FirstSeenAtLTE(runStarted),
                   assetedge.Or(
                       assetedge.SupersededAtIsNil(),
                       assetedge.SupersededAtGT(runStarted),
                   ),
               ).
               All(ctx)
           inputs := make([]OLDataset, 0, len(edges))
           for _, e := range edges {
               inputs = append(inputs, OLDataset{Namespace: t.Namespace, Name: e.FromAsset})
           }
           // Outputs: single dataset = the run's asset
           outputs := []OLDataset{{Namespace: t.Namespace, Name: run.AssetName}}
           // columnLineage facet (D-18 mapping) — SAME point-in-time predicate.
           colEdges, _ := t.ent.ColumnEdge.Query().
               Where(
                   columnedge.ToAssetEQ(run.AssetName),
                   columnedge.FirstSeenAtLTE(runStarted),
                   columnedge.Or(
                       columnedge.SupersededAtIsNil(),
                       columnedge.SupersededAtGT(runStarted),
                   ),
               ).
               All(ctx)
           if len(colEdges) > 0 {
               fields := map[string]any{}
               for _, ce := range colEdges {
                   refs, _ := fields[ce.ToColumn].([]map[string]any)
                   refs = append(refs, map[string]any{
                       "namespace": t.Namespace,
                       "name":      ce.FromAsset,
                       "field":     ce.FromColumn,
                   })
                   fields[ce.ToColumn] = refs
               }
               outputs[0].Facets = map[string]any{
                   "columnLineage": map[string]any{"fields": fields},
               }
           }
           // Run facets — code_hash via processing_engine facet
           runFacets := map[string]any{}
           if run.FinishedAt == nil {
               return RunEvent{}, fmt.Errorf("openlineage: run %s has nil finished_at", runID)
           }
           return RunEvent{
               EventType: "COMPLETE",
               EventTime: run.FinishedAt.UTC().Format(time.RFC3339),
               Run:       OLRun{RunID: runID.String(), Facets: runFacets},
               Job:       OLJob{Namespace: t.Namespace, Name: run.AssetName},
               Inputs:    inputs,
               Outputs:   outputs,
               Producer:  Producer,
               SchemaURL: SchemaURL,
           }, nil
       }

       func (t *DefaultTranslator) TranslateAsset(ctx context.Context, asset string, since time.Time) ([]RunEvent, error) {
           runs, err := t.ent.Run.Query().
               Where(runent.AssetNameEQ(asset), runent.StateEQ("succeeded"), runent.FinishedAtGTE(since)).
               Order(ent.Asc(runent.FieldFinishedAt)).
               All(ctx)
           if err != nil { return nil, err }
           out := make([]RunEvent, 0, len(runs))
           for _, r := range runs {
               ev, err := t.TranslateRun(ctx, r.ID)
               if err != nil { continue } // skip individual failures, log via slog
               out = append(out, ev)
           }
           return out, nil
       }

                   return
               }
               writeJSON(w, http.StatusOK, result)
           }

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTP client → REST handler | Untrusted JSON body + query parameters cross here; auth middleware verifies JWT but body content/depth/asset-name strings are still attacker-controlled |
| REST handler → ent/sqlc → PostgreSQL | Trusted within process; parameterized queries enforce no SQL injection |
| OpenLineage export → external consumer | Outbound — exports lineage metadata; if the endpoint is unauthenticated, lineage is leaked |
| schema_changes ack mutation → DB | Touches RLS-protected ack columns; only governance role allowed |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-07-01 | D (Denial of Service) | GET /v1/lineage/impact?depth=N | mitigate | Hard cap 25 enforced in `impactHandler`; depth>25 returns HTTP 400 with `type=depth_exceeded`. Wave 6 `impact.Analyze` returns `ErrDepthExceeded` which the handler translates. CTE itself caps via `WHERE depth < @max_depth` (Wave 6). Defense-in-depth: handler check + library check + SQL CTE depth bound. |
| T-04-07-02 | T (Tampering) | GET /v1/lineage/impact?asset=… | mitigate | Asset name validated against `^[a-zA-Z0-9_.\-]{1,256}$` regex at handler entry; sqlc parameterized queries prevent SQL injection regardless. Defense-in-depth: handler regex + sqlc parameterization. |
| T-04-07-03 | E (Elevation of Privilege) | POST /v1/schema/changes/:id/ack | mitigate | `auth.RequireRole("governance")` middleware in router group. Non-governance role → 403. Phase 5 will replace string-match with Casbin policy without changing route placement. |
| T-04-07-04 | E (Elevation of Privilege) | PATCH /v1/assets/:name/metadata, PATCH /v1/assets/:name/columns/:col/metadata | mitigate | Same as 07-03 — RequireRole("governance"). |
| T-04-07-05 | I (Information Disclosure) | GET /v1/lineage/export?format=openlineage | mitigate | Endpoint mounted inside JWT-protected group (`auth.Middleware`). Unauthenticated request → 401. Lineage is governance-sensitive; export is read-by-default (any authenticated user) which matches the read access for impact/timeline. If a deployment needs governance-only export, swap the mount block in router.go with one line — interface preserved. |
| T-04-07-06 | D (Denial of Service) | PATCH /v1/assets/:name/metadata with massive tags array | mitigate | 1MB body limit at router level (already present, line 38 of router.go). | T-04-07-10 | T (Tampering) | Cross-asset metadata write (PATCH for asset A but body contains data for asset B) | mitigate | Asset name is taken ONLY from URL path parameter via `chi.URLParam(r, "name")`; request body has no asset field. No path → body cross-talk possible. |
